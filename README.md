# Предобработка и категоризация исторических данных о рынке видеоигр (Python / Pandas)

## Бизнес-задача
Подготовка, глубокая очистка и категоризация исторических данных о продажах видеоигр за период с 2000 по 2013 год включительно. Цель - сформировать чистый датасет для последующего анализа продуктовых метрик.

## Описание данных
Анализируется датасет объемом 16 956 строк, содержащий информацию о продажах игр, платформах, жанрах и оценках (критиков, пользователей, рейтинга ESRB).

## Стек технологий
* **Язык:** Python
* **Библиотеки:** Pandas
* **Навыки:** Очистка данных от строковых артефактов (`to_numeric`), заполнение пропусков на основе группировок, работа со стилями (`str.replace`, `str.lower`), автоматическая категоризация непрерывных величин через `pd.cut()`.

## Ключевые этапы предобработки
1. **Изменение заголовков:** Все наименования столбцов автоматически приведены к стандарту `snake_case`.
2. **Восстановление типов данных:** Из числовых столбцов удалены текстовые артефакты-заглушки (`tbd`, `unknown`). Столбцы продаж и оценок переведены в формат `float`, а годы выпуска - в `int`.
3. **Обработка пропусков:** 
   * Единичные пропуски в критических полях (`name`, `genre`, `year_of_release`) удалены.
   * Пропуски в региональных продажах (`eu_sales`, `jp_sales`) заполнены средними значениями в зависимости от игровой платформы и года релиза.
   * Отсутствующий возрастной рейтинг ESRB выделен в отдельную категорию `UNKNOWN`, чтобы сохранить данные для анализа.
4. **Устранение дубликатов:** Удалены скрытые дубликаты в жанрах (из-за разного регистра), удалено 235 явных строк-дублей.
5. **Контроль качества:** Суммарные потери данных в ходе очистки составили 3.02% (512 строк).

## Результаты фильтрации и категоризации
* Сформирован чистый актуальный срез данных за период 2000-2013 гг. объемом **12,781 строк**.
* Проведена сегментация игр по оценкам пользователей и критиков.
* Выявлен топ-7 игровых платформ по количеству релизов за исследуемый период.
